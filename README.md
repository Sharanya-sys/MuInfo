# MuInfo

MuInfo is a smart museum companion web app that lets visitors scan artworks to instantly learn about them. It includes guided tours, interactive maps, and offline support.

## Features
- AI-based artwork recognition
- Multimedia info pages
- Interactive museum map
- Curated guided tours
- PWA offline access
- Multilingual support

## Tech Stack
- React + Tailwind CSS
- Node.js + Express
- Google Cloud Vision API
- Firebase or MongoDB
